---
title: リモートコントロール
description: ロックをリモートで制御
---

Wi-Fiゲートウェイでは、以下のことができます：
- どこからでもドアの施錠/解錠
- ロック状態をリアルタイムで確認
- 通知を受け取る


### ステップ1

ゲートウェイ機器を購入します。リモート解錠機能を使用するには、まずゲートウェイ機器を購入する必要があります。この機器は，施錠をインターネットに接続し，Welockアプリを使用してリモートで制御できます。

!['sadf'](img.png)

### ステップ2

ゲートウェイとロックとWelockアプリをバインドします。ゲートウェイを購入したら，ロックとWelockアプリにバインドする必要があります。この手順は简单で，数分で完了します。


### ステップ3
Wi-Fiリモートモードに切り替えましょう。ゲートウェイをバインドした後，Welockアプリの右下に 있는Wi-Fi Boxボタンをクリックすると，Wi-Fiリモートモードに切り替えられます。ボタンの上には信号強度が表示されます。
