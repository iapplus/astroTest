---
title: Verbindungsprobleme beheben
description: Häufige Verbindungsprobleme und Lösungen
---

Wenn Sie Probleme beim Verbinden haben, versuchen Sie folgende Lösungen:

- Überprüfen Sie Ihre Wi-Fi-Signalstärke
- Stellen Sie sicher, dass Sie 2,4 GHz Wi-Fi verwenden
- Starten Sie Ihren Router neu
- Setzen Sie das Wi-Fi-Gateway zurück
